Features

Multilayer menu

If you want to create a menu that contains parent-child relationships, you just need to create folders. simpler-paper automatically generate menus based on hierarchical relationships.

Example

source:

|____docs
| |____install
| | |____install.1.md
| | |____install.2.md
| |____problems
| | |____answer.1.md
| | |____answer.2.md
| |____paper.config.json

target:

Sort

Both files and folders support manually specifying weights, you just need to add a prefix to them.
Prefixes are automatically removed after compilation.

Example
source: 

|____docs
| |____install
| | |____9_install.1.md
| | |____5_install.2.md
| |____problems
| | |____answer.1.md
| | |____answer.2.md
| |____paper.config.json

target:

Code highlighting

Adding highlight.js files and highlight.css files will automatically open the code highlighting
You can download and download these files here: highlightjs
